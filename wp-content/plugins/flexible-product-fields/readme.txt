== Description ==

= Free WooCommerce Product Add-Ons Manager =

Customize and personalize your products with the power of WooCommerce product fields and add-ons. Enable customers to customize products with additional options such as **text fields, checkboxes, radio buttons, dropdowns** and more.

Customers will be able to add **t-shirts colors, jewellery engravings, messages on mugs, extra paid insurance or gift packaging**, etc. Right on the product screen, directly above or below the add to cart button.

= All Features of Flexible Product Add-Ons =

* 7 add-on types for creating unlimited input fields.
* Add add-on groups above or below WooCommerce add to cart button.
* Assign field groups to selected products.
* Add-Ons visible in the cart, checkout and admin.
* Required or optional fields.
* Manage labels, values, placeholders and CSS classes for each field.
* Support for WPML and Polylang.
* Duplicate add-on groups.
* Save field groups as drafts (handy when you want to hide add-on group for some time without deleting it).
* Add fixed or percentage prices to add-on (PRO).
* Assign add-on groups to all products or selected categories (PRO).
* Product add-ons conditional logic (PRO).

Simplicity is key. There are many bloated fields plugins. We decided to use a simple design as much as possible to create a really flexible fields editor.

= Product Add-Ons Placement Options =

Show add-on groups above or under the *add to cart* button. Assign to all products, selected products or selected categories. It's your choice.

The add-ons will seamlessly integrate with your theme with minimum CSS added by the plugin. Apart from just selecting options the plugin will also display a clear options prices (if added) and totals.

== Installation	 ==

You can install this plugin like any other WordPress plugin.

1. Download and unzip the latest release zip file.
2. Upload the entire plugin directory to your /wp-content/plugins/ directory.
3. Activate the plugin through the Plugins menu in WordPress Administration.

You can also use WordPress uploader to upload plugin zip file in menu Plugins -> Add New -> Upload Plugin. Then go directly to point 3.
